# Modern Bank Management System

A modern and user-friendly bank management system built with HTML, CSS, and JavaScript. This project demonstrates a clean and intuitive interface for managing bank accounts and transactions.

## Features

- **Dashboard**: View total balance and individual account balances
- **Account Management**: Manage multiple accounts (Savings and Checking)
- **Money Transfer**: Transfer money between accounts
- **Transaction History**: View and filter transaction history
- **Responsive Design**: Works on both desktop and mobile devices

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- JavaScript (ES6+)
- Font Awesome Icons

## Getting Started

1. Clone the repository or download the files
2. Open `index.html` in your web browser
3. Start managing your accounts!
